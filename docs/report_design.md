# 报告格式设计

## 质量检测要求

- 标题 > 3 字符
- 内容 > 200 字符才走 GLM-5 分析
- 失败文章记录到 `logs/quality_check/`

## 报告结构

```markdown
# 📊 价值投资日报 - {日期}

## 一、概览

- 今日新增：X 篇
- 有效分析：Y 篇（内容>200字）
- 无效文章：Z 篇（短状态/评论）

## 二、优先级分类

### 🔴 必读（3-5篇）
- 高质量长文，深度分析

### 🟡 值得关注（5-10篇）
- 有价值的观点和分析

### 🔵 参考（剩余）
- 短状态、预告类

## 三、文章集合

### 🔴 必读文章

#### 1. {标题}
- 作者：{author}
- 链接：{url}
- 字数：{count}
- 相关股票：{stocks}

**GLM-5 分析：**
- 主题归类：{category}
- 核心观点：
  1. {point1}
  2. {point2}
  3. {point3}
- 价值投资评估：{alignment}，安全边际：{margin}
- 投资启示：{insights}
- 一句话总结：{summary}

---

（其他文章同理）

## 四、相关股票汇总

| 股票 | 出现次数 | 文章 |
|------|----------|------|
| 中海油 | 3 | [链接1], [链接2], [链接3] |
| 携程 | 2 | [链接1], [链接2] |

## 五、今日总结

- 主要话题：{topics}
- 投资建议：{advice}
```

## 分析 Prompt 设计

```python
ANALYSIS_PROMPT = """你是一位价值投资研究专家，请深度分析以下投资文章：

## 文章信息
标题：{title}
作者：{author}
字数：{word_count}

## 正文
{content}

## 分析要求

请提供详细的结构化分析：

1. **主题归类**（行业分析/公司研究/投资理念/宏观经济/其他）

2. **相关股票**（提及的具体股票，格式：股票名(代码)）

3. **核心观点**（3-5条，每条50字以上）
   - 观点1：...
   - 观点2：...
   - 观点3：...

4. **价值投资评估**
   - 是否符合价值投资原则：是/否/部分
   - 安全边际评估：高/中/低/不确定
   - 详细分析（100字以上）

5. **投资启示**（2-3条，每条30字以上）

6. **一句话总结**（30字以内）

请以 JSON 格式返回。
"""
```
